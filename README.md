# Robot Dog Mechanical Design
This project presents a mechanical design of a robot dog using Tinkercad

The design includes:
- Main body structure
- Four mechanical legs
- Servo motors for leg movement
- Joints for motion
- Walking direction
- Stable body structure

The robot uses a simple walking method where the legs move sequentially to maintain balance and stability.
A preliminary torque calculation was made for one leg joint using:

Torque = Force × Distance

Assuming the moving leg part has a mass of 0.1 kg and the distance from the joint is 0.08 m:

Force = 0.1 × 9.81 = 0.981 N  
Torque = 0.981 × 0.08 = 0.078 N.m

Overall, this design represents a simple robot dog model that demonstrates the basic principles of mechanical movement, servo control, torque, and stability.
